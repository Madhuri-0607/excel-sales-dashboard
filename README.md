## Project Overview
This project is an interactive Excel dashboard created to understand and track the monthly sales performance of Awesome Chocolates.  
The goal of the dashboard is to turn raw sales data into clear, easy-to-understand insights that help evaluate business performance at a glance.

## What This Dashboard Shows
The dashboard highlights the most important sales metrics and trends, including:
- Overall sales revenue and profit
- Number of boxes sold and shipments made
- Sales performance across different regions
- Contribution of individual salespeople

## Key Metrics Tracked
- Total Sales
- Total Boxes Sold
- Total Shipments
- Total Profit

## Insights & Analysis
Using the dashboard, users can:
- Compare sales performance across countries and regions
- Understand how different product categories contribute to total sales
- Identify top-performing salespeople and areas that need improvement
- Quickly spot trends and patterns in monthly sales data

## Tools & Skills Used
- Microsoft Excel
- Pivot Tables and Pivot Charts
- Slicers for interactive filtering
- Conditional Formatting
- Dashboard design and data organization
